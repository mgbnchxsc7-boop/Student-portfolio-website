# Student Portfolio Website
This project was created as part of the level 5 Diploma in web application.

## Project Overview
This project is a static front-end website created as part of a Level 5 Diploma
in Web Application Development. The site demonstrates the use of HTML5, CSS3,
and Bootstrap to create a structured, multi-page website.

## Purpose and Value
The website provides a simple portfolio layout that introduces the developer,
presents skills, and allows visitors to access contact information.

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- Git & GitHub

## Site Structure
The website contains three main pages:
- Home
- About
- Contact
All pages are connected using a consistent navigation menu.

## Version Control
Git and GitHub were used to track changes during development.

## Deployment
The website is deployed using GitHub Pages.
